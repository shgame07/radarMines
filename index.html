Solo Minesとは

8×8の盤面に隠された5個の爆弾を、マスを開いたときに表示される「距離」の手がかりだけで見つけ出す、1人用の推理モードです。

スコア計算

  • GUESS成功：+200点 / マス
  • GUESS失敗：-150点 / マス
  • 爆弾を踏む：-100点 / 個
  • 未開放マス：+5点 / マス（開かなかったマスにもボーナスがつく）
  • 爆弾を1つも爆発させず、GUESSで5個すべてを言い当てると完全制覇ボーナス+777点。
  • タイムボーナス：確定した時点の残り秒数の半分がスコアに加算される（0.5などの端数もそのまま加算・早くクリアするほど有利）。

基本ルール

  • 盤面は 8×8、爆弾は 5個。
  • 爆弾は最初にクリックしたマスと同時に配置される。そのマスには爆弾が置かれず、そこからチェビシェフ距離3以上離れた位置にのみ生成される。
  • 爆弾同士は縦・横・斜めを含む8方向に隣接しないように配置される。

距離表示

マスを開くと、そのマスから最も近い爆弾までの距離が数字で表示される。距離はチェビシェフ距離（縦・横・斜めをすべて1マスとして数える距離）。

表示は 1〜4。4以上離れている場合はすべて「4」と表示される。

距離4の自動開放

距離「4」のマスを開くと、そのマスを中心とした半径2マス（5×5）の範囲内にある、まだ開いていない距離4のマスだけを自動で開く。

距離1〜3のマスや爆弾は自動では開かない。また、この自動開放は連鎖しない（自動で開いたマスからさらに自動開放が起きることはない）。

爆弾を踏んだ場合

  • 爆発演出とともにスコア-100。
  • その爆弾は「発見済み」となり、以後クリックできない。
  • GUESSできる最大数が1減る。
  • ゲームは続行するが、5個すべての爆弾を爆発させると即ゲームオーバーになる。

GUESSモード

  • GUESSボタンを押すとGUESSモードに入り、爆弾だと思うマスに旗を立てられる（自由に付け外し可能）。一度GUESSモードに入るとプレイには戻れないので、開くマスを選ぶ段階でよく考えよう。
  • 旗を立てられる数の上限は「5 − 爆発済み爆弾数」。
  • GUESSモード中はタイマーが停止する。じっくり考えてから確定してOK。
  • もう一度GUESSボタンを押すと確定。確定後は元に戻れない。

タイマーとランキング

制限時間は480秒（8分）。最初のマスをクリックした時点でスタートする。プレイ中に時間切れになると自動でGUESSモードに入り、その時点の旗の状態でそのまま確定する。

ランキングはスコアが高い順。同スコアの場合は残り時間が多い方が上位、それも同じなら同順位になる。

攻略のコツ

  • 距離「1」が出たマスの周囲は特に危険。複数マスの距離情報を突き合わせてから開こう。
  • 距離「4」のマスは自動で周囲が開けやすいので、うまく使うと安全に情報を稼げる。
  • GUESSモードはタイマーが止まるので、確定を押す前に何度も見直そう。
  • すべて開き切らなくても未開放ボーナスがある。怪しいマスは無理に開けず、GUESSに賭けたほうが得なこともある。
  • 完全制覇ボーナスを狙うなら、爆発を1回もしないことが絶対条件。

Dual Minesとは

2人のプレイヤーがまったく同じ盤面で同時にプレイし、スコアを競うオンライン対戦モードです。基本的なゲームルールはSolo Minesと同じですが、スコア計算はタイムボーナスを含まない点が異なります（勝敗は残り時間ではなくスコアで決まるため）。

スコア計算

  • GUESS成功：+200点 / マス
  • GUESS失敗：-150点 / マス
  • 爆弾を踏む：-100点 / 個
  • 未開放マス：+5点 / マス（開かなかったマスにもボーナスがつく）
  • 爆弾を1つも爆発させず、GUESSで5個すべてを言い当てると完全制覇ボーナス+777点。
  • Solo Minesと違い、タイムボーナスは無い（残り時間は勝敗判定にのみ使われる）。

部屋を作る・参加する

  • 1人が「部屋を作る」を押すと5桁のルームコードが発行される。
  • もう1人がそのコードを入力して「参加する」を押すとマッチ成立。
  • ランダムマッチングは無く、コードを知っている人同士だけで対戦する。

盤面生成とスタート

マッチ成立後、同一の盤面が生成され（論理的に一意確定できる盤面か検証したうえで）、両プレイヤーへ配布される。その後「3・2・1・START!」のカウントダウンが表示され、START!の表示と同時に480秒のタイマーが始まる。

プレイ内容

距離表示・自動開放・爆発・GUESS・スコア計算はソロプレイと完全に同一のルール。ただし盤面は両プレイヤーで同じものを使う。相手の盤面や途中経過・途中スコアは表示されない。

相手のGUESSと20秒ルール

  • どちらかがGUESSを確定した瞬間、相手に「相手がGUESSしました」という通知が表示され、同時に20秒カウントダウンが始まる。
  • この20秒間は通常通りプレイ可能（マスを開く・爆弾を踏むこともできる）。
  • 20秒以内にGUESSしなければ、その時点の状態で自動的にGUESSが確定される。
  • 20秒タイマー中も480秒タイマーは止まらない。480秒が先に終了した場合はそちらが優先され、その時点で強制的にGUESSが確定する。

通信切断・勝敗判定

  • 試合中に通信が切れた（一定時間応答が無くなった）プレイヤーは敗北となる。
  • 勝敗は「①スコア → ②残り時間」の順で判定。完全に一致した場合は引き分け。

試合後

  • 試合終了画面では、勝敗・自分のスコア・相手のスコア・スコア内訳が表示される。
  • 両プレイヤーが「リマッチ」を押すと、新しい盤面で同じ部屋のまま再戦できる。
  • 「ホームへ戻る」でいつでも退出できる。

Solo Minesとの違い（このモード特有の設定）

  • 盤面・距離表示・自動開放（距離4以上のマスで発動）はSolo Minesと完全に同じ。
  • タイムボーナスが無い（残り時間はスコアには加算されず、勝敗判定にのみ使われる）。
  • 相手がGUESSを確定すると、そこから20秒の猶予後に自動的にこちらもGUESSが確定する。
  • 通信が切れた場合、相手の勝ちになる（Solo Minesには無い概念）。

Open Minesとは

2人が1つの盤面を完全に共有しながら交互にマスを開ける対戦モードです。「あと1マス開けたい、でもその情報は相手にも渡ってしまう」という読み合いが最大の魅力です。

スコア計算

  • GUESS成功：+200点 / GUESS失敗：-150点 / 爆弾を踏む：-100点。
  • 盤面を共有するため、未開放マスによるボーナスは無い。タイムボーナス・完全制覇ボーナスも無い。
  • スコアが高い方の勝ち。同点なら引き分け。

マッチングと盤面生成

部屋の作成・参加はDual Minesと同じ（5桁のルームコード）。マッチ成立後、論理的に一意確定できるか検証した盤面が生成され、両プレイヤーへ配布される。「3・2・1・START!」の合図で試合開始。

ターン制

  • 先攻・後攻はランダムに決まる。
  • 各プレイヤーには持ち時間180秒があり、自分のターン中だけ減っていく（相手のターン中は減らない）。
  • 自分のターンでは「①未開放マスを1つ開く」か「②GUESSを宣言する」のどちらかを選ぶ。マスを開いたらターン終了、相手に手番が移る。
  • 片方が開いたマスはもう片方にも即座に反映される（盤面は完全共有）。距離表示・自動開放・爆発などの基本ルールはSolo Minesと同じ。

GUESSフェーズ

  • 自分のターンでGUESSを宣言した瞬間、試合はGUESSフェーズへ移行する。
  • GUESSフェーズでは両プレイヤーが同時に（お互い見えない状態で）旗を配置する。
  • 旗の設置には30秒の制限時間があり、超えると自動でその時点の状態のまま確定する。
  • GUESSフェーズ中は持ち時間(180秒)は両者とも停止する。
  • 両者が確定すると答え合わせが行われる。

持ち時間切れ

自分のターン中に持ち時間180秒を使い切ると、強制的にGUESSフェーズへ移行する。

試合後

  • 試合終了画面では、勝敗・自分のスコア・相手のスコア・スコア内訳が表示される。
  • 両プレイヤーが「リマッチ」を押すと、新しい盤面で先攻後攻を決め直して再戦できる。
  • 「ホームへ戻る」でいつでも退出できる。

Solo Minesとの違い（このモード特有の設定）

  • 距離表示・自動開放（距離4以上のマスで発動）の基本ルールはSolo Minesと同じ。
  • 盤面を2人で完全共有し、交互にターンを行う点がSolo Minesと異なる。
  • 未開放マスボーナス・タイムボーナス・完全制覇ボーナスは無い（盤面を共有するため）。
  • Solo Minesの480秒タイマーの代わりに、持ち時間180秒（自分のターンのみ消費）とGUESSフェーズ30秒という独自の時間制約がある。
  • 通信が切れた場合、相手の勝ちになる（Solo Minesには無い概念）。

SENTINEL MINESとは

盤面に地雷を仕込むGuardianと、それを暴くHunterに分かれる非対称の2ラウンド制対戦です。2ラウンドで役割を交代し、合計スコアが高い方が勝ち。情報戦・心理戦・ブラフが中心のモードです。

Mineの価値の決定方法

Guardianは通常Mineを1〜5個配置できます。配置したMineの総数によって、すべてのMineの価値が決まり、この価値がゲーム中すべてのスコア計算で共通して使われます。

  • 1個配置：1個あたり1000点
  • 2個配置：1個あたり500点
  • 3個配置：1個あたり333点
  • 4個配置：1個あたり250点
  • 5個配置：1個あたり200点

Core Mineは配置数に関係なく600点固定。

その他のスコア（開放マス）

  • Hunterが最後まで開けなかったマス：Hunter +5点／1マス
  • Hunterが開けたマス：Guardian +5点／1マス

フルオープンボーナス

Hunterが一度も爆発させず、Core Mine・Mineすべてを正しくGUESSできた場合：Hunter +777点。

時間ボーナス

  • Hunter：残り時間をそのまま加算（例：残り300秒 → +300点）
  • Guardian：経過時間（360−Hunter残り時間）をそのまま加算（例：Hunter残り300秒 → Guardian +60点）

Guardian側のスコア計算（すべて加算式）

  • ① Core Mineを守り切る：HunterがGUESSできなかった場合 +600点
  • ② Mineを守り切る：HunterがGUESSできなかったMineについて、そのMineの価値をそのまま獲得（例：Mine2個配置→1個だけ発見された→残り1個分 +500点）
  • ③ HunterがMineを踏む：踏まれたMineの価値の50%を獲得（1000→+500 / 500→+250 / 333→+166 / 250→+125 / 200→+100）
  • ④ HunterがCore Mineを踏む：Core Mine価値600点の50% +300点
  • ⑤ Hunterが開けたマス：+5点／1マス
  • ⑥ 時間ボーナス：経過時間（360−Hunter残り時間）をそのまま加算

Hunter側のスコア計算（すべて加算式）

  • ① Core MineをGUESS成功：2×2の4マスのうち1マスでも的中すれば成功 +600点
  • ② MineをGUESS成功：発見したMineの価値をそのまま獲得（1000 / 500 / 333 / 250 / 200点）
  • ③ Mineを踏む：踏んだMineの価値の50%を減点（1000→-500 / 500→-250 / 333→-166 / 250→-125 / 200→-100）
  • ④ Core Mineを踏む：Core Mine価値600点の50% -300点
  • ⑤ 未開放マス：+5点／1マス
  • ⑥ フルオープンボーナス：一度も爆発させずCore Mine・Mineすべてを正しくGUESSできた場合 +777点
  • ⑦ 時間ボーナス：残り時間をそのまま加算

スコア設計コンセプト

このモードは単純なスピード勝負ではなく、Core Mineを守る／見つける、Mineの配置数をどうするか考える、相手にミスを誘発する、少ない情報で推理することを重視したゲームデザインです。

Mineの価値は「配置数」で変化するため、Guardianは配置数自体を戦略として選択でき、Hunterは「踏まない」「少ない情報で見抜く」ことにも高い価値が生まれます。

盤面とラウンド進行

  • 盤面は8×8。距離表示のルールはSolo Minesと同じチェビシェフ距離だが、数字の上限は1〜6（6以上はすべて「6」）。
  • 距離「5」以上のマスを開くと、周囲5×5の未開放の距離5以上マスが自動で開く（連鎖はしない）。Solo Minesの自動開放（距離4以上）より1段階広い。
  • Round1: Player A(部屋を作った人)がGuardian、Player Bがhunter。Round2はGuardian/Hunterが入れ替わる。

Guardianフェーズ（制限時間60秒）

  • まずCore Mine（2×2固定）を1つ配置する。盤面内に完全に収まる位置にのみ置ける。
  • 続けて通常のMineを1〜5個、好きな数だけ配置する（距離の制約は無く、隣接していてもよい。Core Mineや他のMineと重ならなければOK）。
  • Core Mineを置き、Mineを最低1個置くとREADYできる。60秒経過時は自動でREADYになる。

Hunterフェーズ（制限時間360秒）

  • 距離のヒントを頼りにマスを開いていく。開いたマス・爆発・GUESSの様子はすべてGuardianにリアルタイムで見られている（ただしスコアはゲーム終了まで非公開）。
  • GUESSの流れはSolo Minesと同じ：GUESSボタン→旗設置→GUESSボタン→答え合わせ。
  • GUESSできる数は最大6個。Core Mine・Mineどちらに使うかは自由（例：Core1個+Mine5個、Core4マス全部+Mine2個、など）。Core Mineは2×2のうち1マスでも旗を立てれば成功になる。
  • 時間切れになると、その時点の旗の状態で自動的にGUESSが確定する。

ヒントカード（Hunter専用・1試合1回）

  • 📡 Radar Scan：使用すると3〜5マス（ランダムで決定）まで自由にマスを覗ける10秒間のモードに入る。安全マスは10秒経過後すべて未開放に戻る（純粋な偵察用）。ただし爆弾を踏んだ場合は通常通り確定でカウントされる。
  • 🧭 Border Intel：盤面の一番外側の行・列にあるMineの数を教えてもらう（Core Mineも外周に触れていれば1個としてカウント）。
  • 🟢 Safe Reveal：安全なマスをランダムで4マス、そのまま通常開放する（永続的）。
  • 📶 Signal Boost：Core Mineから最も遠い、まだ開いていない安全マスを1マス、そのまま通常開放する（複数該当する場合はランダムで1つ）。

ラウンド進行・試合後

  • Hunterフェーズ終了後、Hunterの盤面・Mine・Core Mine・GUESSの正誤が公開される（この時点ではスコアは表示されない）。
  • Round1終了後は役割交代してRound2へ。Round2終了後、両ラウンドのスコア内訳・合計スコア・勝敗がすべて公開される。
  • 両プレイヤーが「リマッチ」を押すと、新しい盤面で再戦できる（役割は再度Round1から）。

Solo Minesとの違い（このモード特有の設定）

  • 盤面は8×8で同じだが、距離の上限が4ではなく6（表示は1〜6）。
  • 自動開放の発動条件が距離4以上ではなく距離5以上（Solo Minesより1段階広い）。
  • Mineの配置はSolo Minesのようなランダム生成ではなく、Guardianが手動で行う。
  • スコア計算は全く別物：Mine/Core Mineの発見・守り・踏み・時間経過それぞれに専用の計算式があり、Guardian/Hunterで内容が異なる（詳細は上の「スコア計算」参照）。
  • ヒントカードというSolo Minesには無い専用要素がある（Hunterのみ、1試合1回）。
  • GUESSできる最大数は6個（Solo Minesは5個）。Core Mine・Mineどちらに使うかは自由。
  • 通信が切れた場合、相手の勝ちになる（Solo Minesには無い概念）。
